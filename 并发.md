# 并发
1.Reentranlock怎么实现？

- Sync:

    - nonfairTryAcquire

        如果stat=0，cas操作设置stat为1，设置持锁线程
        
        否则，如果重入则加1

    - tryRelease

        先判断是否是持锁线程，然后设置stat为getStat-1

- NonfairSync:

    - tryAcquire调用nonfairTryAcquire
- FairSync:

    - tryAcquire
        如果stat=0，**先判断队列中是否有其他线程等待**，如果没有cas操作设置stat为1，设置持锁线程
        
        否则，如果重入则加1
